# Ibexa REST remote API

This bundle adds REST remote API to [Ibexa DXP](https://www.ibexa.co/products) and Ibexa Open
Source. See [the official documentation](https://doc.ibexa.co/)
or [REST API Reference](https://ezsystems.github.io/ezplatform-rest-reference/) to learn more.
